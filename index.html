JavaScript(JS)

a front-end programming language.

Operators

mathematical functions used to do math.

Variables

Containers used to store information. They allow for a way to label data with a descriptive name.

Variable Naming Conventions

  • Begin with a letter
  • Case sensitive (myNumber !== myNUMBER)
  • Avoid vague letters or initials

Functions

Blocks of code that perform an action and return a result. They may take arguments.

Methods

Actions run on pieces of data.

Arguments

Information fed into methods to help it perform its action.

Paremeters

Variables assigned to an argument

Return

A value returned from a method or function.

Chaining Methods

Calling a method directly on the return value of another method.

Strings

Data type that represents the exact text of whatever is enclosed in quotes.

Booleans

Data type with a value either true or false.

Undefined

An object (such as a variable) without a defined value.

NaN

A numerical value- stands for "Not a Number"

Escape

"\" A character that prevents the browser from reading your text as code

Alert

alert() - a JS function that pops up a dialog box

Comments

Text added to code that is not interpreted by the browser as code. It is used to help organize your code.

jQuery

A JavaScript library that makes it easy to make webpages interactive

Attributes

Code added to an HTML element to modify it (class, id, etc)
